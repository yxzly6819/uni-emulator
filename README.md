# 大学模拟器

一个关于中国大学教育的讽刺模拟游戏。四年、八个学期、十六次关键决策——在选课、精力分配和随机事件中，走向属于你的结局。

**[在线游玩](https://yxzly6819.github.io/uni-emulator/)**

---

## 版本更新

### v0.4 (2026-07-19)

- 学期操作改为两步向导：先选课 → 再分配精力，界面铺开
- 延毕阈值从 GPA < 1.0 提高到 < 2.0
- 大四选择自学活动即自动设置考研备考标记
- 中期反馈和期末总结界面大幅美化

### v0.3 (2026-07-19)

- 每次半学期事件触发概率降至 50%，上下半学期不再必定都有事件
- 期末绩点由上半学期和下半学期投入档位的**平均值**计算（不再只看下半学期）
- 支持 GitHub Pages 部署

### v0.2 (2026-07-18)

- 绩点公式重写为加性模型：`ability × aIdx + effort × eIdx ± random`
- 政治课改为能力权重极低、努力权重极高——死磕即满分
- 新增 5 个随机事件，扩充可重复事件池
- 标题页精简；移除事件分类标签
- 每次选课时提醒剩余政治课数量

### v0.1 (2026-07-18)

- 初版：完整游戏循环（选课 → 精力分配 → 事件 → 结算 → 8学期 → 结局）
- 13 门课程（EECS 5门 + 法学 4门 + 政治 4门），22 个随机事件
- 8 种结局（保研/考研上岸/考研失败/就业三档/休学/延毕）

## 技术栈

React + TypeScript + Tailwind CSS · Vite · React Context + useReducer

## 本地运行

```bash
npm install
npm run dev
```

## 部署

push 到 `master` 分支，GitHub Actions 自动构建部署到 GitHub Pages。
